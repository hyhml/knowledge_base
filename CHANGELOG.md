# 版本说明

## v0.4.5 - 2026-07-14

共享数据读取修复版本。

### 修复

- 访客打开网页时改为直接读取 `./data/knowledge.json`
- 避免未登录 GitHub API 限流导致页面回退到旧的浏览器缓存
- 只有协作者保存数据时继续使用 GitHub API
- 如果已有 Token 但 API 读取失败，会自动回退到公开数据文件

## v0.4.4 - 2026-07-14

典型例题扩充版本。

### 新增

- 将 21 个知识点的典型例题统一扩充到每条 4 题
- 例题覆盖概念判断、基础计算和常见易错场景
- 同步更新共享数据和内置样例数据

### 校验

- 21 个知识点均包含 4 道典型例题
- 每道例题均包含题目和解法

## v0.4.3 - 2026-07-14

全量知识点内容补齐版本。

### 新增

- 按 `二次函数图像与最值` 的内容模板补齐剩余 20 个知识点
- 每个知识点增加 3 个 B 站推荐视频
- 每个推荐视频增加推荐理由
- 每个知识点扩充为 8 条必会结论
- 每个知识点扩充为 8 条常见错误

### 校验

- 21 个知识点均包含讲解、推荐视频、必会结论、常见错误
- 共记录 63 条推荐视频引用
- 同步更新共享数据和内置样例数据

二次函数知识点内容实验版本。

### 调整

- 扩充 `二次函数图像与最值` 的必会结论到 8 条

知识点推荐视频补充版本。

### 新增

- `二次函数图像与最值` 增加 3 个 B 站推荐视频链接

## v0.4.0 - 2026-07-14

两栏知识库布局版本，删除中间筛选结果列表，减少重复导航。

### 新增

- 左侧 Sidebar 合并搜索、掌握状态筛选、难度筛选、知识结构树和协作同步
- 知识树升级为“学科 → 模块 → 章节 → 知识点”层级
- 共享数据增加 `subject` 字段，当前默认为“数学”

### 调整

- 删除中间“筛选结果列表”区域
- 删除旧的 `KnowledgeList`/卡片列表渲染逻辑
- 右侧详情区自适应占据剩余主空间
- 搜索和筛选后，左侧知识树直接显示匹配结果
- 点击左侧知识点后，右侧直接展示详情，不再经过二级列表跳转
- 编辑表单中的“单元”文案调整为“章节”

## v0.3.0 - 2026-07-14

知识结构图谱版本，优化左侧导航和右侧知识点阅读布局。

### 新增

- 左侧改为树状知识结构图谱：模块 → 单元 → 知识点
- 树状图谱由知识点数据中的 `module` 和 `unit` 自动生成，避免手写层级造成关系错误
- 点击模块、单元、知识点可同步定位右侧内容
- 筛选结果增加计数显示

### 调整

- 将搜索、掌握状态、难度筛选移动到顶部工具区
- 右侧详情区获得更大阅读空间
- 筛选结果为空时不再显示旧知识点详情
- 前置知识点跳转会同步定位到对应树节点

## v0.2.0 - 2026-07-14

协作编辑版本，支持使用 GitHub 仓库 JSON 文件作为共享数据源。

### 新增

- 新增 `data/knowledge.json`，作为共享数学知识点数据文件
- 网页打开时自动加载 GitHub 仓库中的共享知识点数据
- 新增“协作同步”面板
- 支持协作者输入 GitHub Token 后保存修改到仓库
- 保存前检查远程文件版本，减少多人编辑时的覆盖风险
- README 增加 GitHub Pages 分享方式和协作者 Token 权限说明

### 说明

- 普通访问者可以查看知识点
- 协作者需要仓库写权限和 `Contents: Read and write` Token 才能保存
- Token 不写入仓库，只保存在当前浏览器会话中

## v0.1.0 - 2026-07-14

首个可用版本，主攻数学知识点库。

### 新增

- 建立静态 Web 应用结构：`index.html`、`styles.css`、`app.js`
- 添加高三数学基础知识点样例数据
- 支持模块、掌握状态、难度筛选
- 支持关键词搜索
- 支持知识点详情展示
- 支持前置知识点关联与跳转
- 支持新增、编辑、删除知识点
- 支持使用浏览器本地存储保存修改
- 支持恢复内置样例数据

### 暂不包含

- 错题库
- 讲解记录库
- 学生账号
- 云端同步
- 自动复习提醒
- 练习卷生成
